## Getting Started

Clone the repo:
```bash
git clone https://github.com/zuni-developer/zuni-developer.github.io.git
cd zuni-developer.github.io
```

Install dependencies:
```bash
npm install
```

Run locally:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Deploy to GitHub Pages:
```bash
npm run deploy
```
